Code is beautiful!

Visualizing source code can help programmers and team leads to better understand their code. Creating useful and engaging visualizations of code can be challenging though. With this project, we want to change that by providing an open-source collection of source code visualizations that can be used in the browser. All of charts are realized with either D3 oder Three.js and are provided under an Affero GPL license, so you can freely reuse them in your own projects.

Please feel free to contribute your own visualizations.
